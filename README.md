# Alke Wallet

## Descripción
Aplicación de billetera digital desarrollada en Java que permite:
- Gestión de fondos
- Conversión de moneda

## Tecnologías
- Java
- JUnit
- Maven

## Ejecución

1. Clonar repositorio
2. Ejecutar:

mvn clean install
mvn exec:java

## Funcionalidades
- Crear cuenta
- Depositar dinero
- Retirar dinero
- Convertir moneda
